# blocks

组件(vue)可视化布局编辑器，核心功能：
1. 属性编辑。
2. 指令编辑。
3. 样式编辑。
4. 事件增删改。
5. 变量(data)、函数(methods) 增删改。
6. 支持通过编写配置导入不同的组件库，内置 Vant、Antd 组件库。
