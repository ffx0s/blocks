# blocks

组件(vue)可视化布局编辑器，核心功能：

1. 组件属性、指令、样式、事件等增删改。
2. 支持添加变量(data)、函数(methods)。
3. 支持编写JS表达式。
4. 实时编辑所见即所得，导出可维护的源码。
5. 支持通过编写配置导入不同的组件库，内置 Vant、Antd 组件库。
